Program Functionalities:

Budgeter is a program that allows users to map their expenditures and create budgeting plans in an attempt to become more mindful of where their money is going. The following is what it is capable of:

Expenses
- Add transactions with amount, date/time, merchant, note, category and the currency.
- Auto-refreshing UI.
- Browse and search transactions by name and category.
- Sorts transactions by days and months.

Budgets
- Create budgets with name, limit, currency and start/end date.
- Create two types of budgets (overall and category specific).
- Display live progress showing current expenditure against total budget using a progress bar.

Overview
- Shows total spending with month and year as long with date progression.
- Displays active budget plans.
- Shows categories with highest spending.
- Ability to add expense from this page.

Insights
- Shows spending by category.
- Shows spending over time using charts.

Settings
- Ability to change currency.
- Appearance: System / Light / Dark.

Error Handling
- Preventive UI: Disables save option until inputs are valid, no empty variables etc.
- Retraceable Options: Allows for deletion of data.
- Utilises try blocks with appropriate fail messages to avoid crashes and notify user.

Getting Started:
1. Open project in Xcode
2. Run on simulator or device
3. Upon opening program automatically creates an account, example expense and premade budget
4. Use add expense to log new expenses
5. Use add budget to create new custom budgets
6. Use insights tab to reflect on data
7. Use settings to change appearances or clear data
